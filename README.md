# TechshipProto

A Vue.js 3 web application for financial management platform designed for sports teams and small organizations.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Run the development server:
```bash
npm run dev
```

## Build

Build for production:
```bash
npm run build
```

## Preview

Preview production build:
```bash
npm run preview
```

## Tech Stack

- Vue 3
- Vue Router 4
- Vite
